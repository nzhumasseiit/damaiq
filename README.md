
## DamaIQ - шашки которые вы знаете с детства, только умнее.

**Русские шашки с AI-разбором, рейтингом городов и двумя режимами правил.**

DamaIQ это веб-платформа для игры в русские шашки. Можно играть против AI, смотреть историю ходов, получать разбор партии через AI Coach и соревноваться в городском рейтинге по Казахстану.

🔗 **[damaiq.vercel.app](https://damaiq.vercel.app)**

## Почему DamaIQ


Большинство сайтов для шашек в интернете используют американские правила: дамка ходит на одну клетку, обычная шашка не бьёт назад. Для игроков из Казахстана и постсоветского региона это ощущается неправильно, ведь это не те шашки в которые играли в детстве.

DamaIQ реализует правила русских шашек: дамка ходит и бьёт на дальнюю дистанцию, обычная шашка может бить назад, взятия обязательны, а превращение в дамку во время цепочки взятий обрабатывается по правилам.

Движок написан на TypeScript с нуля и отделён от UI. Компоненты отвечают за интерфейс, а вся игровая логика находится в отдельном модуле и покрыта тестами.

## Почему название DamaIQ

В русских и казахских шашках дамкой называют шашку, которая дошла до последней линии и получила полный ход по диагонали.

**DamaIQ** это Дама + IQ: игра про тактику, внимание и умение думать на несколько ходов вперёд.

## Что внутри

### Игровой движок

- Полные правила русских шашек: диагональные ходы, обязательные взятия, цепочки взятий, дамки с дальним ходом, превращение во время взятия
- Нотация ходов: `c3-d4`, `e5:g7:h8`
- Unit-тесты на Node test runner
- Отдельный слой логики, не завязанный на React-компоненты

### Два режима правил

**Строгий режим**: классические русские шашки. Если есть взятие, игрок обязан бить.

**Свободный режим / Еркін режим**: режим для обучения и спокойной игры. Взятие необязательно, поэтому новичкам проще разобраться с доской и движением фигур.

### AI-противник

- **Лёгкий**: случайные ходы, подходит для первой игры
- **Средний**: предпочитает взятия и длинные цепочки
- **Сложный**: minimax с alpha-beta pruning, глубина 6

### AI Coach

После партии можно запросить разбор через Claude API. Coach анализирует историю ходов и объясняет:

- где было упущено взятие
- какой ход открыл дамку сопернику
- какие решения были сильными
- что можно улучшить в следующей партии

Разбор выполняется на серверной стороне, поэтому API-ключ не попадает во frontend.

### Городской рейтинг

Игрок выбирает город перед первой партией, а результат идёт в общий рейтинг. Идея простая: не только личная партия, но и небольшое соревнование между городами. Это старый добрый вайб (кто сильнее?) в современном интерфейсе.

Поддерживаемые города: Алматы, Астана, Шымкент, Қарағанды, Атырау, Ақтөбе, Павлодар, Өскемен, Тараз.

## Стек

Frontend: Next.js 16, React 19, TypeScript, Tailwind CSS 4  
AI Coach: Anthropic Claude API, server-side  
База данных: Supabase, anonymous auth, profiles, game history  
Leaderboard: Supabase Realtime  
Анимации: canvas-confetti  
Деплой: Vercel

## Планы

- Мультиплеер по ссылке: Supabase Realtime channels, room code типа `DAMA-4829`, без регистрации
- Stripe billing: реальные платежи для Pro-подписки
- Replay партий: просмотр партии ход за ходом
- Тренировочные задачи: позиции, где нужно найти лучший ход
- Международные шашки: отдельный режим 10x10
- PWA: установка приложения на телефон
- Турниры: городские онлайн-турниры с bracket-системой

## Запуск локально

```bash
git clone https://github.com/yourusername/damaiq
cd damaiq
npm install
```

Создай `.env.local`:

```env
ANTHROPIC_API_KEY=your_key
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
```

```bash
npm run dev
```

Без env-переменных приложение работает в offline-режиме: localStorage вместо Supabase, mock-ответы вместо AI Coach.

### Supabase setup

1. Создай проект на [supabase.com](https://supabase.com)
2. Запусти `supabase/schema.sql` в SQL Editor
3. Authentication → Providers → Anonymous → включи

## Тесты

```bash
npm test
```

Покрыты: обычные ходы, взятие назад, обязательное взятие, цепочки взятий, дальний ход дамки, взятие дамкой, превращение, превращение во время взятия с продолжением.
